# Javascript Starter pack


## Lancer votre bot

### partie classée
```
npm run start -- -r -s <TOKEN>
```

### partie non-classée
```
npm run start -- -s <TOKEN>
```

Vous pourrez ajouter la logique de votre code dans [src/bot.js](src/bot.js). C'est le seul fichier que vous avez besoin de modifier.
